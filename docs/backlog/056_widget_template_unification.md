# 056. jbexport + jbtp 위젯 템플릿 통합 — organization 표시 정책

**상태**: 🟢 신규 (다음 주 명확화 후 진행)
**제안일**: 2026-05-10
**발견**: 5/10 jbtp 위젯 fix (053) 완료 시점 — jbexport 위젯과 표시 형태 불일치 인지
**우선순위**: 중 (마감 단계, UX 영향 작음 — 그러나 사이트 추가 시마다 불일치 누적 우려)

## 배경

5/10 마라톤 동안 jbexport (049/050/051) + jbtp (053) 두 source 의 위젯을 동일 정책 (공지 제외, 사이트 표시순 매칭) 으로 정렬했음. 그러나 organization 표시 형태는 두 사이트가 다름:

| Source | organization 분포 | 위젯 노출 |
|---|---|---|
| **jbtp** | 100% `전북테크노파크` (단일 기관) | "전북테크노파크" 통일 표기 |
| **jbexport** | 9종 (047/6/4/3/3/2/1/1/1, 050 백필 결과) | 다양 — 공고마다 다른 기관명 |

→ 사용자 체감: jbtp 위젯은 기관명 중복으로 의미 없고, jbexport 는 기관 다양성이 정보 가치. 통합 위젯 템플릿에서 어떻게 처리할지 정책 결정 필요.

## 현황

### jbtp (053 후)
- 모든 row organization='전북테크노파크' (사이트 자체가 단일 기관 게시판)
- 위젯 5건 = "전북테크노파크" 5번 반복 표시 → noise

### jbexport (050 후)
- 9종 기관 분포:
  - 47 (재)전북특별자치도 경제통상진흥원
  - 6 코트라 전북지원본부
  - 4 한국무역협회 전북지역본부
  - 3 전주시 / 3 전북특별자치도
  - 2 전북지방우정청
  - 1 한국무역보험공사 전북지사 / 1 전주상공회의소 / 1 (재)전북바이오융합산업진흥원
- 위젯에서 기관명 보이면 사용자가 어떤 기관 사업인지 즉시 구분 가능

## 옵션

### A. source 별 분기 유지 (현 상태)
- jbtp = "전북테크노파크" 표시, jbexport = 각 기관명 표시
- 장점: 정보 손실 없음
- 단점: 위젯 템플릿 분기 유지, 시각적 일관성 부족

### B. organization 표시 일괄 제거
- 위젯에서 기관명 컬럼/필드 자체 제거 → 제목 + 마감일 + source 라벨만
- 장점: 템플릿 통합 단순, 모바일 표시 깔끔
- 단점: jbexport 의 기관 다양성 정보 손실

### C. source 라벨로 통합 표시
- 위젯 모두 "[전북수출통합]" / "[전북테크노파크]" / 등 source 라벨 한 줄 + 제목
- jbexport 의 9종 기관명은 상세 페이지에서만 노출
- 장점: 모든 source 동일 템플릿, 시각적 일관성
- 단점: jbexport 기관 정보 1단계 깊이로 이동 (클릭 필요)

## 결정 필요 사항 (다음 주)

1. **jbexport 기관명 다양성**이 사용자에게 실제 정보 가치인지 (사용자 클릭 패턴 / feedback)
2. **모바일 위젯**에서 기관명 표시가 텍스트 ovverflow 발생 빈도
3. 향후 **신규 source 추가 시** (예: bizinfo / kstartup 등 위젯 노출 후보) 어떤 정책이 확장 쉬운지
